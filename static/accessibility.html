Accessibility

  • Pages have sufficient title's (tab title)
  • Used labels in forms where it was possible
  • All page meets the HTML5 standards
  • Background and foreground colors have a sufficient contrast ratio
  • List items (< li >) are contained within < ul > or < ol > parent elements
  • Heading elements appear in a sequentially-descending order
  • Img tags have alt statments
  • Forms have font-end(JS) and back-end(PHP) checks
  • Cross-site scripting (XSS) protection
  • SQL injection protection
  • Unauthorized users cannot access the site
